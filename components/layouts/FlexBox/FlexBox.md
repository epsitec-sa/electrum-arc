# FlexBox

## Usage:

```jsx
<FlexBox direction='' wrap='' justify='' >
  <FlexItem>...</FlexItem>
</FlexBox>
```
### direction=

* row (default)
* row-reverse
* column
* column-reverse

### wrap=

* no (default)
* yes
* reverse

### justify=

* start (default)
* end
* center
* space-a
* space-b

### align-items=

* baseline
